Home

Welcome to the Munetios API documentation. Here you will find information about the available APIs and how to use them.

Feel free to explore the different sections to learn more about each API.

Munetios provides a collection of APIs designed to help developers quickly add beautiful styles and improved HTML structures to their web projects. Our APIs are easy to integrate, require no registration, and are free to use for personal and commercial projects.

  • Beautiful CSS API: Instantly apply modern, visually appealing CSS styles to your website with a single line of code.
  • Better HTML API: Enhance your HTML markup for better readability and maintainability, making your code cleaner and easier to work with.

Getting Started:

  1. Choose the API you want to use from the navigation bar above.
  2. Copy the provided <link> tag and paste it into the <head> section of your HTML file.
  3. Refresh your website to see the changes instantly.

For detailed usage instructions, code examples, and advanced customization options, visit the documentation sections for each API. If you have questions or need support, please refer to our FAQ or contact us through our support channels.

Thank you for choosing Munetios APIs to enhance your web development workflow!

Beautiful CSS API

You can use the "liquid-glass" class name to make transparency effects easily. You can use blur-xs, blur-sm, blur-md, blur-lg, blur-xl, blur-2xl, and blur-3xl to apply different levels of blur to your elements. You can use transparency, transparency-xs, transparency-sm, transparency-md, transparency-lg, transparency-xl, transparency-2xl, and transparency-3xl to apply different levels of transparency to your elements. Also addional is the text-glow class to add a glow effect to text, the shadow classes like shadow-light, shadow-medium, shadow-heavy, and shadow-glow, the hover-zoom class to zoom on hover, the active-scale class to scale on active, and the pulse class to add a pulsing animation. More classes at beautiful.css

Better HTML API

This API provides tools to enhance your HTML coding experience.

Use it to simplify your HTML structure and improve readability.

You can use <card> and <cards> to make row of cards.

You can also use <topbar> to create a top navigation bar.

Also you can use <logo> to create a logo element.
